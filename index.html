Bedienung
• Bauteil wählen, dann aufs Blatt klicken (Shift = mehrfach)
• Bauteil anklicken → Fenster am Bauteil konfiguriert es; ziehen = verschieben
• Mehrfachauswahl: Rahmen aufziehen oder Strg+Klick
• Strg+C kopieren · Strg+V einfügen · Strg+X ausschneiden
• Strg+Z rückgängig · Strg+Y wiederholen
• Leitung: Anschluss (Kreis) anklicken, dann zweiten Anschluss anklicken
• Je Anschluss nur eine Leitung — Verzweigungen über Knoten auf der Leitung
• Abzweig: beim Leitungsziehen auf eine Leitung klicken, oder Doppelklick auf eine Leitung setzt einen Knoten
• Mittlere Leitungssegmente lassen sich mit der Maus verschieben
• R = drehen · X = löschen · Esc/Rechtsklick = abbrechen
• Mausrad = zoomen · mittlere Maustaste = Blatt verschieben
• Schalldämpfer rastet direkt auf Anschlüssen ein
• Bauteil löschen lässt Leitungen als Knoten stehen — neues Bauteil einfach passend daraufsetzen
• Endlagen-Sensor: auf den Zylinder setzen (vorn = ausgefahren, hinten = eingefahren); am Ventil Betätigung „Pneumatisch (Sensor)" wählen
• Simulation: Ventilseite anklicken und halten = betätigen; rastende Betätigungen schalten je Klick um. Strg+Klick hält Taster gedrückt, bis Strg losgelassen wird (mehrere gleichzeitig möglich). Druck-betätigte Ventile bekommen Steueranschlüsse 12/14. Dunkelblaue Leitungen führen Druck.
